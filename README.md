# TempTalk

**TempTalk is a security & privacy focused chat app.**

We encourage the submission of any security vulnerabilities identified in TempTalk products.

For detailed policies and guidelines, please refer to [our security page](
https://github.com/RBPioi/TempTalk-Bug-Bounty/security).

To help us verify and assess your findings efficiently, we recommend submitting your report directly through our [GitHub security page](https://github.com/RBPioi/TempTalk-Bug-Bounty/security).

If you prefer email submission, please see the [Email Submission](https://github.com/RBPioi/TempTalk-Bug-Bounty/security#email-submission) section.

For any inquiries, please contact us via:
- Email: security@chative.com
